1.Explique a diferença entre programação orientada a objetos e programação estruturada.
R: Programação orientada a objetos oferece aos desenvolvedores um padrão de encapsulamento, construção de classes
tudo agrupado em objetos. 
A estruturada o foco está nas funções e nos procedimentos que devem ser seguidos.

3.Explique o que é um banco de dados relacional.
R: Banco de dados relacional são bancos que conseguem fazer "ligações de 1 para muitos ou de muitos para 1 (1:N).
Um exemplo é um autor que pode escrever varios livros, 1 para muitos.

4.Dê um exemplo de um algoritmo de busca.
R: Um algoritmo de busca é uma idéia que vc pode constituir ao percorrer uma lista de forma linear até encontrar
um item desejado como por exemplo fazer um for para percorrer um array até encontrar o que foi definido na busca.

5.Explique o que é um algoritmo de ordenação.
R: Algoritmo de ordenação é quando na busca você impoe uma regra através de uma logica como por exemplo
ordenar numeros de forma crescente ou separar vogais de consoantes

6.Explique a diferença entre uma variável e uma constante.
R: Variaveis como let e var podem sofrer alteração de valor durante a função ou globalmente, constantes (const) recebe um valor
e esse valor não pode ser modificado.

7.Dê um exemplo de uma estrutura de controle de fluxo.
R: Estrutura de controle de fluxo envolve como se dá a sequência de execução das instruções do programa, 
isso envolve loops como for e while e/ou estruturas de repetição como o for e condicionais com if/else.

8.Explique o que é um algoritmo.
Algoritmo é a maneira como se organiza um pensamento, os passos,  uma logica para chegar a um resultado, objetivo ou
conclusão.

9.Dê um exemplo de um tipo de dado primitivo.
São blocos de instruções simples que envolvem strings formadas por chars, booleanos, numbers (int, parseFloat)

10.Explique o que é um objeto.
Objeto são tipos de dados que contém atributos, que constituem propriedade com valor
